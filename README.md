## About

Eco-Socks is a landing page designed to showcase sustainable and comfortable socks made from eco-friendly materials. It aims to attract customers interested in environmentally conscious products and highlight the benefits of choosing Eco-Socks. The project is built using HTML and CSS, focusing on a clean and responsive design.

This landing page provides information about the materials used, the manufacturing process, and the comfort and durability of Eco-Socks. It targets eco-conscious consumers looking for sustainable alternatives in everyday products.

The project utilizes basic HTML structure and CSS styling to create an appealing and informative website. It emphasizes the visual presentation of the product and its eco-friendly aspects.
